# 📝 Aplikasi CAT Simulasi Seleksi Substansi BCKS 2026

Aplikasi berbasis web interaktif (CBT/CAT) yang dirancang khusus untuk membantu persiapan uji kompetensi **Bakal Calon Kepala Sekolah (BCKS)** tahun 2026. Menggunakan pendekatan instrumen terbaru **Situational Judgment Test (SJT)** yang berfokus pada studi kasus kepemimpinan transformasional di sekolah.

## 🚀 Fitur Aplikasi
* **70 Butir Soal Komprehensif:** Soal prediksi akurasi tinggi mencakup 5 kompetensi inti kepala sekolah.
* **Real-time Countdown Timer:** Batasan waktu pengerjaan ujian selama 120 menit murni.
* **Interaktif Dashboard & Peta Soal:** Navigasi mudah lengkap dengan status penanda soal (Terjawab, Kosong, Ragu-Ragu).
* **Diagnosis Hasil Instan:** Kalkulasi akurasi nilai dan analisis kelemahan kompetensi otomatis pasca-ujian.

## 🛠️ Klaster Kompetensi yang Diujikan
1. Kepemimpinan Pembelajaran & Kurikulum Merdeka
2. Perencanaan Berbasis Data (PBD) & Rapor Pendidikan
3. Supervisi Akademik & Coaching Klinis (TIRTA)
4. Manajemen Konflik & Hubungan Masyarakat (Humas)
5. Kewirausahaan & Pengelolaan Aset Sekolah (*Asset-Based Thinking*)

1. Unduh file `index.html` di repositori ini.
